\chapter{Introducción}

En este documento encontramos el resultado final de la práctica \textbf{ETL con PIG} \cite{pig} , enmarcada dentro de la asignatura de Big Data II del máster en Ciencia de Datos de la Universidad de Granada. 

\section{Problema a resolver}
\label{problema}

Se pide diseñar un experimento de carga y extracción de información de una base de datos usando \textbf{PIG}, para realizar consultas en un entorno distribuido Hadoop. PIG es un lenguaje creado para flujo de datos que permite especificar como captar y leer estos flujos. Es útil para:

\begin{itemize}
\item ETL.
\item Analizar datos en bruto.
\item Procesamiento de datos iterativo.
\end{itemize}

La base de datos elegida, es una base de datos de quejas de usuarios de tarjeta de crédito en Estados Unidos, con un total de 65499 muestras con 18 características por cada muestra. 

\section{Objetivos}

Los objetivos de esta práctica podrían resumirse en los siguientes:

\begin{itemize}
	\item Carga de los datos.
	\item Realizar un experimento original.
	\item Resolver el experimento con consultas que incluyan una proyección, selección, agrupamiento y funciones de agregado y resumen sobre los datos.
\end{itemize}
